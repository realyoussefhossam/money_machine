# Money Machine
Solidity Smart Contract & dApp made for Money Machine

## Solidity Smart Contract
A BUSD staking protocol with features such as:
- Progressively increasing APR
- 30-day locking period
- Fees
- Owner functionality
- Referral system
- Coupon system

## The dApp
Built with the old but golden trio: HTML, CSS, JS (web3js, jQuery).

**Link:** https://moneymachine.pro/
